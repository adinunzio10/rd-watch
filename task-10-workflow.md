## Quick Commands:
```bash
# Switch to agent context
task-master use-tag task-10-<agent>

# Get next task
task-master next

# Start working
task-master set-status --id=<task-id> --status=in-progress

# Complete task
task-master set-status --id=<task-id> --status=done
```
